- **线上**：https://codante.github.io/hongkong2026/
- **部署**：push 到 main → GitHub Actions 自动 build + 部署（约 1-2 分钟）。本地不用跑 dev，改完 push 即可。
- **核心事件**：7/25（六）15:00 JCS《万世巨星》下午场 @ 香港文化中心大剧院（约 2h，整趟行程的锚点）

## 架构约定（重要）

### 文件地图
- 🎨 **样式层（可改视觉）**：`theme.js`（色板+字体，全局入口）、`components/*.jsx`、`App.jsx`、`main.jsx`
- 📝 **文案层（Coda 自己改）**：`data/content.js` — 所有行程文字集中于此，按场景命名分块，**不含逻辑**
- 🔢 **数据层**：`data/foods.js`（潮汕美食清单）、`accommodation.js`（住宿）、`transport.js`（交通段）
- 🔒 **逻辑层（轻易别动）**：`lib/itinerary.js`（选方案+算日期，文案全引用 content.js）、`lib/plan.js`（buildLegs/buildStays）

### 硬约定
- 样式 = **内联 style 对象 + theme.js 色板**，无 CSS/Tailwind。沿用这套，别引新框架。
- 改文案只动 `content.js`，不碰 `itinerary.js` 分支逻辑。
- 改完必须 `npm run build` 能过；保持响应式（朋友手机看）。
- `CLAUDE.md` 和 `.claude/` 已 gitignore（含个人信息，勿入公开仓库）。

### 调性
- 暖色衬线美学（COLORS 暖橙+金，serif 标题）。Coda 是设计师，视觉她在意。
- 行程文案走"松弛留白"风：餐食只写"觅食"不定死每顿，待定项用 `？` 标记。
- 交通/住宿数据要诚实：参考价标区间、注明"实时以携程为准"，不编造精确数字。
