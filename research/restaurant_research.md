# 🍽️ 香港美食调研 — Coda 7月行程用

> 调研日期：2026-05-19
> 行程：2天，尖沙咀（香港文化中心看剧7.25）+ 港岛（中环/湾仔/金钟）
> 原则：不吃火锅、潮州菜、台湾菜、拉面（去汕头吃中餐），在香港吃港式点心+西餐+甜品消暑
> 来源：OpenRice、Google、Tripadvisor、Michelin Guide、官网

---

## ☕ 早茶 / 点心

### 逸东轩 Yat Tung Heen ⭐ 强推

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | Google 4.3 · Tripadvisor 4.5 (#60/13,915) · OpenRice 4/4 |
| 🏅 米其林 | ⭐ 一星（2017–2025连续） |
| 💰 人均 | HK$300–500（点心午市偏低） |
| 📍 地址 | B2/F, Eaton HK, 380 Nathan Road, Jordan |
| 🚇 地铁 | 佐敦站 C2 出口步行2分钟 |
| 🕐 营业 | 11:00–15:00, 18:00–23:00 |
| 🍽️ 招牌 | 传统粤式点心（40–50款）、广东海鲜 |
| 📞 订位 | 建议提前，Chope/OpenRice 可在线订 |

**口碑：** 米一星里性价比很高，点心出品稳定上菜快。1920s上海风装修有氛围。午市点心限时90分钟。部分评价提到忙时服务态度波动。

**适合场景：** 看剧当天午餐 / 抵达第一餐。离文化中心非常近。

### 六安居 Luk On Kui

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | OpenRice 约3.5/4（改名后评价重新积累中） |
| 💰 人均 | HK$100–150 |
| 📍 地址 | 2-3/F, 40-50 Des Voeux Rd West, Sheung Wan |
| 🚇 地铁 | 上环站 A2 出口步行5分钟 |
| 🍽️ 招牌 | 推车点心、现点现做 |

**口碑：** 前身是有名的蓮香居，2024年员工接手改名"六安居"。**香港少数仍保留推车点心的茶楼**——阿姨推着车在桌间穿梭，看到什么拿什么，非常传统的饮茶体验。环境比较旧、比较吵、比较混乱，但这就是老香港茶楼的样子。CNN Travel 推荐过。

**适合场景：** 想要"最原汁原味的港式饮茶体验"选这家。如果更在乎舒适度和品质选逸东轩。

## 🥩🍝 西餐正餐

### Bouillon Bistro Parisien ⭐ 强推

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | Google/Tripadvisor 4.5（485条评价） |
| 💰 人均 | HK$380–450（三道菜套餐 $398） |
| 📍 地址 | 6 Pound Lane, Sheung Wan |
| 🚇 地铁 | 上环站步行5分钟 |
| 🕐 营业 | 午餐 12:00–14:30, 晚餐 18:00–22:30 |
| 🍽️ 招牌 | Rossini de Boeuf、Beef Tartare、各种法式经典 |

**口碑：** 4.5分+485条评价，在香港法餐里算非常稳的高分。原帖说"随便点都不踩雷"，从数据看确实如此。法式小酒馆氛围，菜品用有机新鲜食材。三道菜套餐$398性价比不错。

**适合场景：** 去港岛那天的正餐。在上环，可以和六安居/中环其他店串起来。

### 22 Ships

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | Google ~4.3 · OpenRice 新店评价少 |
| 💰 人均 | HK$400–800 |
| 📍 地址 | S109–S113, 1/F PMQ, 35 Aberdeen Street, Central |
| 🚇 地铁 | 中环站 D2 出口步行10分钟 |
| 🕐 营业 | 周二至周日 12:00–15:00, 18:00–23:00（**周一休息**） |
| 🍽️ 招牌 | 章鱼饭、Tortilla($168)、Gazpacho |
| 💡 特别 | **不收服务费** |

**口碑：** 2024年从湾仔搬到中环PMQ新址，环境升级但评价还在积累。Tapas风格适合两人share。不收服务费算良心。但有评价提到服务员暗示加小费，单价不低（生蚝$78/只），两人吃饱可能$1200+。

**适合场景：** 在PMQ里，可以顺便逛创意园区。注意周一休息。

**适合场景：** 和Bouillon二选一，或者如果在上环逛的话可以当备选。

**适合场景：** 如果想要一顿"有海景的fine dining"，这是好选择。但人均$800+含酒水轻松破千，看预算。

## 🍜 快餐 / 简餐

### 水门鸡饭

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | OpenRice 好评为主 |
| 💰 人均 | HK$50–80 |
| 📍 地址 | 尖沙咀金马伦道18号万勤商业大厦地铺 |
| 🚇 地铁 | 尖沙咀站步行5分钟 |
| 🍽️ 招牌 | 水门半只鸡（$108）、泰式鸡饭 |

**口碑：** "性价比封神"的泰式鸡饭小店。鸡皮薄少肥膏、肉质嫩滑，配鸡骨汤。两个人吃半只鸡+饭+汤大概$150搞定。

**适合场景：** 尖沙咀区域随便吃一顿的首选。便宜、快、不踩雷。

## 🍧 甜品 / 消暑（7月必备！）

### Shari Shari 氷屋 ⭐ 强推

| 项目 | 信息 |
|------|------|
| ⭐ 评分 | Google 4.4 |
| 💰 人均 | HK$45–90 |
| 📍 中环店 | 47 Staunton Street, Central |
| 📍 **尖沙咀店** | G/F, 83-85A Chatham Rd South, TST |
| 🕐 营业 | 13:00–22:00（中环）/ 13:00–22:30（尖沙咀） |
| 🍽️ 招牌 | 草莓刨冰配北海道牛乳布丁、焙茶、抹茶、伯爵奶茶口味 |

**口碑：** 北海道人开的日式刨冰店，用日本纯净水制冰，口感绵密细滑。多个口味都好评，料足量大。**在尖沙咀有分店**——看完剧出来散步刚好路过。7月天热，这家是消暑首选。

**适合场景：** 穿插在任何时段。尖沙咀和中环都有店，路过就吃。
