# 餐厅调研 Pilot — 4家

> 调研日期：2026-05-19
> 调研来源：OpenRice、Google、Tripadvisor、Michelin Guide、官网

---

## 1. 逸东轩 Yat Tung Heen

| 项目 | 信息 |
|------|------|
| ⭐ Google | 4.3 (Tripadvisor 4.5/5, #60/13,915) |
| ⭐ OpenRice | 4/4 |
| 🏅 米其林 | 一星（2017–2025 连续获星） |
| 💰 人均 | HK$300–500（点心午市偏低，晚市偏高） |
| 📍 地址 | B2/F, Eaton HK, 380 Nathan Road, Jordan |
| 🚇 最近地铁 | 佐敦站 C2 出口步行2分钟 |
| 🕐 营业时间 | 11:00–15:00, 18:00–23:00 |
| 🍽️ 招牌 | 传统粤式点心、广东海鲜 |
| 📞 订位 | 建议提前订，可通过 Chope/OpenRice 在线订位 |

**口碑摘要：**
- ✅ 米其林一星里性价比算高的，点心午市尤其划算
- ✅ 点心种类丰富（40–50款），出品稳定，上菜快
- ✅ 1920s上海风装修，环境有氛围感
- ⚠️ 午市点心限时90分钟，节假日可能比较赶
- ⚠️ 部分评价提到服务态度波动，忙时可能怠慢

**Coda备注：** 在佐敦，离尖沙咀非常近。适合安排看剧当天的午餐或第二天的早茶。米一星点心午市是性价比之选。

## 2. La Vache!

| 项目 | 信息 |
|------|------|
| ⭐ Google | 4.4 |
| ⭐ OpenRice | 4/4（482条评价） |
| 💰 人均 | HK$398/人（固定套餐价） |
| 📍 地址（中环店） | 48 Peel Street, SoHo, Central |
| 📍 地址（尖沙咀店） | 尖沙咀亦有分店！ |
| 🚇 最近地铁 | 中环站 D2 出口步行8分钟（SoHo店） |
| 🕐 营业时间 | 12:00–15:00, 18:00–22:30 |
| 🍽️ 菜单 | 只有一道菜：沙拉 + USDA Prime肋眼牛排 + 无限续薯条 + 甜品 |
| 📞 订位 | 建议提前订，OpenRice订位送甜品 |

**口碑摘要：**
- ✅ 评价数量多（482条）且分数稳定，踩雷概率极低
- ✅ 概念简单粗暴——不用纠结点什么，坐下就吃
- ✅ 牛排用铁板+蜡烛保温，薯条现炸无限续，甜品（朱古力蛋糕/柠檬挞）好评
- ✅ 法式复古装修，适合两人约会
- ⚠️ 只有牛排一个选择，不吃牛的人没法去
- ⚠️ 算上酒水可能到HK$500+/人

**Coda备注：** 尖沙咀也有店！不一定要跑去中环。两个人吃大概HK$800不含酒。概念有趣但你说了不太想吃牛排，建议作为备选而非首选。

## 3. 22 Ships

| 项目 | 信息 |
|------|------|
| ⭐ Google | 约4.3 |
| ⭐ OpenRice | 新店评价较少（7条），旧湾仔店口碑好 |
| 💰 人均 | HK$400–800 |
| 📍 地址 | Shop S109–S113, 1/F, Staunton Building, PMQ, 35 Aberdeen Street, Central |
| 🚇 最近地铁 | 中环站 D2 出口步行10分钟 |
| 🕐 营业时间 | 周二至周日 12:00–15:00, 18:00–23:00（周一休息） |
| 🍽️ 招牌 | 章鱼饭（原清单称全港Top1）、Tortilla（$168）、Gazpacho |
| 📞 订位 | 建议提前订 |
| 💡 特别 | 无服务费（香港西餐厅少见） |

**口碑摘要：**
- ✅ 2024年从湾仔搬到中环PMQ，新店环境升级，半开放式设计
- ✅ 主厨Antonio Oviedo来自马德里，Tapas风格适合两人share
- ✅ 不收服务费，在香港西餐厅里算厚道
- ⚠️ 新店评价还少，搬家后品质是否稳定有待观察
- ⚠️ 有评价提到服务员会暗示加额外小费（"the food is expensive enough"）
- ⚠️ 单价不低（生蚝$78/只、扇贝$88/只），两人吃饱可能HK$1200+

**Coda备注：** 在PMQ里，去中环的话可以顺便逛逛这个创意园区。Tapas适合两个人分着吃。但人均偏高，7月去的话注意是否是周一（休息日）。

## 4. 麦文记面家 Mak Man Kee

| 项目 | 信息 |
|------|------|
| ⭐ Google | 4.1 |
| ⭐ OpenRice | 约3.5–4/4（评价数多但近年有下滑） |
| 🏅 米其林 | Bib Gourmand（必比登推荐，即"性价比之选"） |
| 💰 人均 | HK$40–60 |
| 📍 地址 | 佐敦白加士街51号地下 |
| 🚇 最近地铁 | 佐敦站 C2 出口步行3分钟 |
| 🕐 营业时间 | 12:00–00:30（营业到凌晨） |
| 🍽️ 招牌 | 鲜虾云吞面、南乳猪手面 |

**口碑摘要：**
- ✅ 70年老店，米其林必比登连续多年推荐
- ✅ 人均$50以内，快进快出，适合不想折腾的一餐
- ✅ 营业到凌晨，看完剧出来还能去吃宵夜
- ⚠️ **近年口碑有下滑趋势**——多条2024–2025评价提到"不如以前""汤底变了"
- ⚠️ 典型老字号问题：环境一般、服务粗犷、夏天可能闷热
- ⚠️ 作为"港式面家"的代表，可能正好是你说的那种"一堆避雷"的类型

**Coda备注：** 离佐敦/尖沙咀很近，营业到凌晨是大优势（看完剧出来宵夜）。但口碑下滑的信号值得注意——如果要吃港式云吞面，可能需要再对比几家。也可以当作"走过路过随便吃一碗"的选择，别抱太高期待。
